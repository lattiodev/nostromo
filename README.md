# Nostromo

## Installation & Run

```bash
npm install
npm run dev
```

## Usage

1. **Get Tier** - `/user/settings/my-tier/change`

2. **Create Project** - `/projects/new`

3. **Vote** - `/projects/dao-voting`

4. **Create Fundraising** - `/projects/create-fundraising/:projectIndex`

5. **Investing** - `/projects/fundraising/:fundraisingIndex`

6. **Claiming** - `/projects/fundraising/:fundraisingIndex`
